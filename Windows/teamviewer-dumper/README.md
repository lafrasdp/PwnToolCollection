# Dump TeamViewer ID and password from memory
Tested on TeamViewer 13 (latest): Win10, Win8.1, Win7x64, Win7x32

Installation:
------

```bash
pip3 install frida
```
You can simply run it or compile as a standalone binary: `pyinstaller .\tv-dumper.py --onefile`
